# MicroStrategy Mosaic Masterclass

A premium, interactive 25-slide presentation for MicroStrategy Developers, Architects, and Administrators.

## 🚀 Features
- **Mosaic Navigation**: Visual tile-based overview of all content.
- **Deep Technical Content**: Spanning modeling, development, and administration.
- **Premium UI**: Glassmorphism, dark mode, and dynamic entrance animations.
- **Responsive**: Works on desktop, tablet, and mobile.

## 📁 Project Structure
- `index.html`: Technical content and application structure.
- `style.css`: Design system and layout logic.
- `script.js`: Navigation framework and interactivity.
- `assets/`: Custom generated thematic backgrounds.
- `GEMINI.md`: Project context and manifest.
- `system.md`: Design system and architecture deep-dive.
- `agent.md`: Maintenance guide for AI agents.

## 🛠️ Usage
Simply open `index.html` in your browser.
- **Arrow Right / Space**: Next Slide
- **Arrow Left**: Previous Slide
- **'M' Key / Grid Icon**: Toggle Mosaic View

## 🎯 Target Audience
- **Architects**: Learn about the Semantic Graph and Logical Modeling.
- **Developers**: Master Metrics, Filters, and Dossiers.
- **Administrators**: Control security, performance, and platform monitoring.
