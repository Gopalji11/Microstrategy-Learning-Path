# MicroStrategy Mosaic Masterclass - Project Context

## Overview
This project is a high-end, 25-slide technical presentation designed to train users on MicroStrategy Architect, Developer, and Administrator roles. It utilizes a "Mosaic" layout system for modern, rapid navigation.

## Core Directives
1. **Premium Aesthetics**: Maintain the glassmorphism and dark mode theme.
2. **Technical Accuracy**: Ensure slides reflect current MicroStrategy platform (MST One) capabilities.
3. **Interactivity**: Every slide transition and interactive element is driven by the internal JS framework.

## Tech Stack
- **Structure**: Semantic HTML5.
- **Styling**: Vanilla CSS3 (Custom Design System).
- **Icons**: Lucide Icons (CDN).
- **Logic**: Vanilla Javascript (State-based navigation).

## Deployment
This is a static SPA (Single Page Application). Load `index.html` via any modern browser or host on a simple static server.
